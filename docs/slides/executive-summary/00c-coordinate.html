Agent Orchestration

Agents delegate, coordinate, and resume automatically

The Metaphor: Project Manager

A good project manager doesn't do everything themselves. They break big tasks into smaller ones, delegate to specialists, track progress, and handle handoffs. Agent Orchestration works the same way. One agent coordinates others, each doing what they're best at.

Delegation
Complex tasks break into manageable parts handled by specialists
Parallel execution
Sub-agents can work simultaneously for faster results
Automatic coordination
Parent knows when children finish, no manual tracking
Resumption
Interrupted work continues where it left off, no lost progress
