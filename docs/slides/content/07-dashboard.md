## What You See

A web-based UI showing everything happening in your agent system:

### Sidebar: Active Sessions and Runners

- **Active Sessions** - See which agents are running, idle, or completed
- **Runners** - Monitor connected Agent Runners and their status

### Main Panel: Event Timeline

A **live** feed of all events across your agents:

- **TOOL** events - File reads, glob searches, API calls
- **MSG** events - Agent messages and reasoning
- **STATUS** events - Session starts, completions, errors

### Navigation Tabs

- **Sessions** - View and manage agent sessions
- **Events** - Real-time event stream
- **Blueprints** - Configure agent definitions
- **Chat** - Interact with agents directly
